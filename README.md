## üéì Technical Notes

### Timezone Handling

- Uses `toLocaleString()` with `timeZone` parameter
- Supports all IANA timezone identifiers
- Automatic DST (Daylight Saving Time) handling
- GMT offset calculated dynamically
- Timezone names from `Intl.DateTimeFormat`

### Performance Optimization

- Efficient DOM updates (50ms intervals)
- RequestAnimationFrame for animations
- Debounced resize handlers
- Lazy audio context initialization
- Optimized CSS transitions

### Browser APIs Used

- **Intl API**: Timezone formatting
- **Wake Lock API**: Screen keep-alive
- **Local Storage API**: Settings persistence
- **Fullscreen API**: Immersive mode
- **Web Audio API**: Sound synthesis
